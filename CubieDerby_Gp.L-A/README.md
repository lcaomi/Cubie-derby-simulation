# 团子竞速 Monte Carlo 模拟 — Group L-A

CLI 程序，通过蒙特卡洛模拟计算团子竞速游戏中每个普通团子的胜率、名次概率分布、平均名次及 99% 置信区间。

无第三方依赖，仅使用 Python 标准库。

## 快速开始

```bash
python simulate_dango.py --n 1000000 --seed 20260514
```

## 命令行参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `--n` | `1000000` | 模拟次数 |
| `--seed` | `20260514` | 随机种子（可复现） |
| `--workers` | CPU 核心数 | 并行进程数 |

## 1M 模拟结果

24 核，~3 秒完成 1,000,000 次模拟。

| 团子 | 胜率 | 99% CI | 平均名次 |
|------|------|--------|---------|
| 陆赫斯 | 21.18% | [21.07%, 21.28%] | 3.46 |
| 莫宁 | 17.34% | [17.24%, 17.43%] | 3.94 |
| 菲比 | 16.45% | [16.35%, 16.54%] | 3.51 |
| 弗诺诺 | 16.41% | [16.32%, 16.51%] | **2.40** |
| 长离 | 15.12% | [15.03%, 15.21%] | 4.07 |
| 玲奈 | 13.51% | [13.42%, 13.59%] | 3.62 |

**结论：**
- **胜率最高**：陆赫斯 (21.18%) — 绿色格 +3 加成最强
- **平均名次最好**：弗诺诺 (2.40) — 底层 +3 稳定输出，46.5% 概率拿第 2 名（常带人冲线）
- **表现最稳定**：弗诺诺 (名次标准差 1.03)

## 架构

- 角色以整数索引（0-5 普通，6 哈基布），数组查表 O(1)
- 棋盘为 32 个 Python list，表示每格堆叠（底→顶），利用 C 实现的 list 操作
- `ProcessPoolExecutor` 并行：每 worker 独立 batch + 唯一种子
- 纯标准库：`argparse`、`math`、`random`、`multiprocessing`
