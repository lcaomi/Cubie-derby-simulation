# 技术路线

基于 Numba JIT + 多进程并行对"团子快跑"桌游进行千万级蒙特卡洛模拟，计算各团子的获胜概率、期望名次及 99% 置信区间（Wilson score interval）。

项目包含 **B组** 和 **C组** 两套独立的模拟实现，分别对应不同团子阵容和技能规则。

## B组 vs C组

| | B组 | C组 |
|------|------|------|
| **团子** | 千咲、莫宁、琳奈、爱弥斯、守岸人、柯莱塔 | 奥古斯塔、尤诺、弗诺诺、长离、今汐、卡卡罗 |
| **游戏结束** | 任意团子到达31号位立即结算 | 全部6名团子到达31号位 |
| **哈基布行动** | 第4回合起，每轮普通团子行动结束后单独行动 | 第4回合起加入随机行动序列 |
| **骰子生成** | 每轮预生成所有团子基础骰子（千咲需要） | 各自掷骰，无需预生成 |
| **排名依据** | 位置（越靠近31越靠前）→ 堆叠高度 | 累计前进距离 → 堆叠高度 |

## 游戏规则概要

### 地图

32个点位（0-31），环形排列。普通团子从 **0号位** 正向移动（0→1→…→31），哈基布从 **31号位** 逆向移动（31→30→…→0）。

| 类型 | 点位 | 效果 |
|------|------|------|
| 绿色 | 2, 10, 15, 22 | 移动组再前进 1 格 |
| 红色 | 9, 27 | 移动组后退 1 格 |
| 黑色 | 5, 19 | 随机重排该格所有普通团子（不含哈基布） |

### 通用规则

- 普通团子掷骰 1~3，移动时带动自身及上方团子（下方不受影响）
- 移动组到达目标后置于该格堆叠顶部
- 哈基布掷骰 1~6，从31号位逆向移动
- 哈基布遇到普通团子时堆叠于最底部
- 绿色/红色效果**不连锁触发**
- 特殊点位效果不连锁（只触发一次）
- 同格内堆叠越靠上排名越靠前
- 哈基布不参与最终排名

### B组团子技能

| 团子 | 技能 |
|------|------|
| **千咲** | 若本轮基础骰子等于所有普通团子中的最小值，额外 +2 步 |
| **莫宁** | 固定骰子循环 3→2→1，每次行动推进 |
| **琳奈** | 60% 双倍 / 20% 无法移动 / 20% 正常 |
| **爱弥斯** | 经过15号位后，传送至前方最近普通团子的堆叠顶部（每场1次） |
| **守岸人** | 骰子仅出 2 或 3（各50%） |
| **柯莱塔** | 28% 概率双倍前进 |

### C组团子技能

| 团子 | 技能 |
|------|------|
| **奥古斯塔** | 处于堆叠最顶端时本轮不行动，下轮最后行动 |
| **尤诺** | 经过中点后，传送其他团子到自己所在格（每场1次） |
| **弗诺诺** | 处于堆叠最底层时移动 +3 |
| **长离** | 下方有团子时 65% 概率下轮最后行动 |
| **今汐** | 头顶有团子时 40% 概率移至该格顶部 |
| **卡卡罗** | 处于最后一名时移动 +3 |

## 技术实现

- **Numba JIT** (`@njit(cache=True)`)：核心游戏循环编译为机器码
- **链表式棋盘**：`tile_bot` / `above` / `below` 数组，O(1) 堆叠操作
- **SplitMix64 RNG**：内联零开销随机数生成
- **多进程并行**：`multiprocessing.Pool`，23核约 50 万局/秒
- **Wilson score interval**（99% 置信区间）

## 快速开始

### B组

```bash
cd CubieDerby_Gp.B-A
python simulate.py --trials 10000000 --seed 42
```

### C组

```bash
cd CubieDerby_Gp.C-A
python simulation.py 10000000 --workers 8
```

### 参数

| 参数 | 说明 |
|------|------|
| `--trials` / `-t` | 模拟次数 |
| `--seed` / `-s` | 随机种子 |
| `--workers` / `-w` | 并行进程数 |
| `--batch` / `-b` | 批大小 |

## 规则歧义与假设

详见各组 README。通用假设：

| # | 歧义 | 假设 |
|---|------|------|
| 1 | 哈基布起始回合 | "第三回合后" → **第4回合** |
| 2 | 绿色/红色连锁 | **不连锁触发** |
| 3 | 普通团子超过31 | **视为到达31** |
| 4 | 黑色点位重排 | 只重排**普通团子**，哈基布不动 |
| 5 | 普通团子是否带动哈基布 | **不带动** |
