【Claude 工作指南：团子赛跑模拟系统】

一、任务目标
1. 构建完整游戏规则模型
2. 使用蒙特卡洛方法模拟不少于1000000次
3. 计算每个团子的：
   - 胜率（第一名概率）
   - 99%置信区间
4. 输出结构化统计结果

--------------------------------------------------

二、游戏地图

1. 点位范围：0 - 31（环形）
2. 特殊点位：
   绿色点（2, 10, 15, 22）：
      抵达后所有团子向前移动1格（含堆叠）
   红色点（9, 27）：
      抵达后所有团子向后移动1格（含堆叠）
   黑色点（5, 19）：
      随机重排该点位所有普通团子的堆叠顺序

--------------------------------------------------

三、基础规则

1. 初始状态：
   团子从0开始按顺序排列

2. 每轮流程：
   (1) 随机排列行动顺序
   (2) 按顺序执行：
       - 投骰子（1~3）
       - 执行移动
       - 触发技能
       - 处理堆叠
       - 处理地图效果

3. 移动规则：
   - 堆叠底层带动上层移动
   - 上层移动不影响下层

4. 排名规则：
   同一格中，从上到下排名

--------------------------------------------------

四、胜利条件

满足：通过0点并最终到达31点
第一个满足条件的团子为第一名
若有堆叠，按堆叠顺序排名

--------------------------------------------------

五、特殊单位：哈基布

1. 初始位置：31（不参与排名）
2. 第3回合后加入行动顺序
3. 行动方向：31 → 0
4. 骰子：1~6

特殊规则：
- 遇到正向移动团子 → 压到最底层
- 如果被正向团子超过 → 重置回31
- 参与所有地图效果

--------------------------------------------------

六、团子技能

奥古斯塔：
  若在堆叠顶层：
  - 本回合不行动
  - 下回合最后行动

尤诺：
  每局一次
  当经过中点并且前后存在其他团子：
  - 将这些团子传送至自己位置
  - 保持原排名顺序

弗洛洛：
  若在堆叠底部：
  - 移动额外+3格

长离：
  若下方有团子：
  - 下回合65%概率最后行动

今汐：
  若上方有团子：
  - 40%概率移动到最上层

卡卡罗：
  若当前为最后一名：
  - 移动额外+3格

--------------------------------------------------

七、模拟要求

1. 模拟次数：
   ≥ 1000000（建议更高）

2. 随机内容：
   - 行动顺序
   - 骰子结果
   - 技能触发
   - 黑点重排

所有必须为均匀随机

--------------------------------------------------

八、标准模拟流程

for 每局游戏:
    初始化状态
    while 未结束:
        打乱行动顺序
        for 每个团子:
            检查技能
            投骰子
            移动
            更新堆叠
            处理地图
        更新哈基布
        判断是否结束

--------------------------------------------------

九、统计方法

1. 胜率：
   p = 获胜次数 / 总次数

2. 99%置信区间：
   CI = p ± 2.576 * sqrt(p(1-p)/N)

--------------------------------------------------

十、输出格式

团子名称 | 胜率 | 99%置信区间

示例：
长离 | 0.2134 | [0.2095, 0.2173]

--------------------------------------------------

十一、校验要求

1. 所有胜率总和 ≈ 1
2. 结果稳定（分批收敛）
3. 无异常值

--------------------------------------------------

十二、常见错误

- 忽略堆叠带动
- 技能触发时机错误
- 哈基布逻辑错误
- 未处理连续特殊地块效果

--------------------------------------------------

十三、推荐结构

GameState
  - positions
  - stacks
  - order
  - round

函数：
  simulate_game()
  run_simulation()
  compute_statistics()

--------------------------------------------------

十四、最终输出要求

必须包含：

1. 模拟总次数
2. 每个团子的胜率
3. 99%置信区间
4. 简要结果分析

--------------------------------------------------
