# 团子赛跑模拟 — Group C-B

使用蒙特卡洛方法 + numba JIT + 多进程并行对团子赛跑游戏进行高性能模拟，计算每个团子的胜率及 99% 置信区间。

## 快速开始

```bash
# 默认 1,000,000 次模拟
python simulate.py

# 自定义模拟次数和工作线程
python simulate.py 5000000 -w 16

# 查看单局详细过程（调试用）
python simulate.py --test
```

依赖：Python 3.10+, numpy, numba

## 角色与技能

| 团子 | 技能 |
|------|------|
| **今汐** | 上方有团子时 40% 概率移至最上层 |
| **长离** | 下方有团子时 65% 概率下回合最后行动 |
| **卡卡罗** | 当前为最后一名时移动额外 +3 格 |
| **弗洛洛** | 在堆叠底部时移动额外 +3 格 |
| **奥古斯塔** | 在堆叠顶层时跳过本回合、下回合最后行动 |
| **尤诺** | 每局一次，经过中点时将其他团子传送至自身位置 |

## 模拟结果（5,000,000 局，99% 置信区间）

| 排名 | 团子 | 胜率 | 99% 置信区间 | 平均排名 |
|------|------|------|-------------|----------|
| 1 | 今汐 | 22.43% | [22.38%, 22.48%] | 3.41 |
| 2 | 长离 | 20.05% | [20.01%, 20.10%] | 3.24 |
| 3 | 卡卡罗 | 17.57% | [17.52%, 17.61%] | 3.40 |
| 4 | 弗洛洛 | 15.60% | [15.56%, 15.64%] | 2.54 |
| 5 | 奥古斯塔 | 12.99% | [12.95%, 13.03%] | 4.49 |
| 6 | 尤诺 | 11.36% | [11.32%, 11.40%] | 3.93 |

| 团子 | 前三率 | 99% 置信区间 |
|------|--------|-------------|

| 卡卡罗 | 48.13% | [48.07%, 48.19%] |
| 尤诺 | 36.74% | [36.68%, 36.79%] |

### 简要分析

- **今汐** 胜率最高（22.43%），其 40% 跳至堆叠顶层的技能可有效规避被底层拖累的风险
- **长离** 紧随其后（20.05%），强制延迟行动的技能在拥挤时优势明显
- **弗洛洛** 尽管胜率仅排第四，但前三率高达 81.40%，是"永远接近冠军"的角色
- **奥古斯塔** 表现最差（12.99% 胜率，49.32% 末位率），堆叠顶层跳过回合的惩罚过重
- **尤诺** 一次性传送不足以与持续性被动技能竞争

## 技术架构

- **numba JIT**：核心仿真循环编译为机器码，采用链表式棋盘表示
- **SplitMix64**：快速均匀伪随机数生成器
- **multiprocessing.Pool**：多进程并行批量运行，充分利用多核 CPU
- **Wilson 分数区间**：99% 置信区间计算

## 命令行参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `sims` | 模拟总次数 | 1,000,000 |
| `-w, --workers` | 工作进程数 | CPU 核心数 - 1 |
| `-b, --batch` | 每批模拟次数 | 50,000 |
| `-t, --test` | 运行单局详细调试模式 | — |
| `--test-n` | 调试模式局数 | 1 |
