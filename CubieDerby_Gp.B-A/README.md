# 团子快跑 — 2026年B组蒙特卡洛模拟

基于 Numba JIT + 多进程并行对"团子快跑"B组桌游进行千万级蒙特卡洛模拟，计算各团子的获胜概率、期望名次及 99% 置信区间。

## 快速开始

```bash
# 默认 1000 万次模拟（多核并行）
python simulate.py

# 自定义模拟次数
python simulate.py --trials 1000000

# 指定随机种子和并行数
python simulate.py --trials 10000000 --seed 42 --workers 8
```

### 参数说明

| 参数 | 短参数 | 默认值 | 说明 |
|------|--------|--------|------|
| `--trials` | `-t` | 10_000_000 | 模拟次数 |
| `--seed` | `-s` | 42 | 随机种子（保证可复现） |
| `--workers` | `-w` | CPU核心数-1 | 并行进程数 |
| `--batch` | `-b` | 50000 | 每批次模拟量 |

## 游戏规则

### 地图

32个点位（0-31），环形排列。普通团子从 **0号位** 出发正向移动（0→1→…→31），哈基布从 **31号位** 出发逆向移动（31→30→…→0）。

移动有界：普通团子到达31后不再前进，哈基布到达0后不再后退。

### 特殊点位

| 类型 | 点位 | 效果（普通团子） | 效果（哈基布） |
|------|------|-----------------|---------------|
| 绿色 | 2, 10, 15, 22 | 移动组再前进 1 格 | 继续逆向移动 1 格 |
| 红色 | 9, 27 | 移动组后退 1 格 | 反向（正向）移动 1 格 |
| 黑色 | 5, 19 | 随机重排该格所有普通团子 | 不参与重排 |

- "移动组"指移动发起者及其上方所有团子
- 绿色/红色效果**不连锁触发**

### 移动规则

- 普通团子掷骰 1~3，正向移动
- 移动时带动自身及上方所有团子一起移动
- 下方团子不受影响
- 移动组到达目标点位后置于现有堆叠顶部

### 骰子生成

每轮开始时**预生成所有普通团子的基础骰子**，用于千咲的最小值比较。技能修正（琳奈双倍、柯莱塔双倍等）在实际移动时应用。

### 游戏结束与排名

**任意普通团子到达31号位时，游戏立即结算。** 排名规则：

1. 同组到达31的团子按堆叠顺序**从上到下**排列名次（上方 = 名次高）
2. 其余普通团子按**当前位置**排序（位置越大越靠前，同格内越靠上越靠前）
3. 哈基布不参与排名

### 哈基布

- 起始于31号位，**第4回合起**每轮在所有普通团子行动**结束后单独行动**
- 逆向移动（31→30→…→0），掷骰 1~6
- 格子内遇到普通团子时永远堆叠于最底部
- 整轮中如有普通团子位置超过哈基布，哈基布传送回31号位

## 团子技能

| 团子 | 技能 | 说明 |
|------|------|------|
| **千咲** | 骰子最小时额外+2 | 若本轮基础骰子等于所有普通团子中的最小值，额外前进 2 步 |
| **莫宁** | 固定骰子循环 | 不随机掷骰，按 3→2→1 循环，每次行动推进一次 |
| **琳奈** | 三重判定 | 60% 双倍 / 20% 无法移动 / 20% 正常 |
| **爱弥斯** | 单向传送 | 经过15号位后（pos≥16），传送至前方最近普通团子的堆叠顶部（每场1次） |
| **守岸人** | 高骰子 | 骰子只会出 2 或 3，各 50% |
| **柯莱塔** | 概率双倍 | 28% 概率以双倍点数前进 |

## 技术实现

- **Numba JIT** (`@njit(cache=True)`)：核心游戏循环编译为机器码，单核约 4.8 万局/秒
- **链表式棋盘**：`tile_bot` / `above` / `below` 数组实现 O(1) 堆叠操作
- **SplitMix64 RNG**：内联零开销随机数生成
- **多进程并行**：`multiprocessing.Pool`，23核约 53 万局/秒
- **Wilson score interval**（99% 置信区间）

## 规则歧义与假设

| # | 规则歧义 | 采用假设 |
|---|---------|---------|
| 1 | 哈基布从第几回合开始行动 | **第4回合**（"第三回合后"） |
| 2 | 哈基布行动顺序 | 每轮所有普通团子行动**结束后**单独行动 |
| 3 | 绿色/红色效果连锁 | **不连锁触发** |
| 4 | 爱弥斯前方无团子时 | 技能**不消耗**，后续回合仍可触发 |
| 5 | 千咲比较对象 | 比较**基础骰子**（技能修正前），非最终移动距离 |
| 6 | 莫宁循环推进时机 | 每次**实际行动时**推进 |
| 7 | 普通团子超过31 | **视为到达31**，立即结算 |
| 8 | 黑色点位重排 | 只重排**普通团子**，哈基布保持不动 |
| 9 | 普通团子是否带动哈基布 | **不带动**（哈基布独立于堆叠移动） |
| 10 | 哈基布移动是否带动上方团子 | **带动**上方团子一起移动 |
| 11 | 千咲并列最小值 | 只要等于最小值即触发（**含并列情况**） |
