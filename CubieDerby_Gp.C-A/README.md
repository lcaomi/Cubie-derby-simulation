# 团子快跑 — 2026年C组第一轮蒙特卡洛模拟

基于蒙特卡洛方法对"团子快跑"桌游进行千万级模拟，计算各团子的获胜概率、期望名次及99%置信区间。

## 项目概述

- **模拟次数**：默认 10,000,000 次
- **置信区间**：99% Wilson score interval
- **引擎**：Python + Numba JIT 编译优化
- **并行**：多进程并行（默认使用 CPU 核心数 - 1）

## 快速开始

```bash
# 默认 1000 万次模拟
python simulation.py

# 自定义模拟次数
python simulation.py 1000000

# 指定并行进程数
python simulation.py 5000000 --workers 4

# 单局详细调试模式
python simulation.py --test

# 多局调试模式
python simulation.py --test --test-n 5
```

## 游戏地图

32个点位（0-31），环形排列。普通团子从 **0号位** 出发正向移动（0→1→…→31），哈基布从 **31号位** 出发逆向移动（31→30→…→0）。

### 特殊点位效果

| 类型 | 点位 | 效果 |
|------|------|------|
| 🟢 绿色 | 2, 10, 15, 22 | 终点为此格的团子向前移动1格（含堆叠团子） |
| 🔴 红色 | 9, 27 | 终点为此格的团子向后移动1格（含堆叠团子） |
| ⚫ 黑色 | 5, 19 | 随机重排该格上所有普通团子的堆叠顺序（不含哈基布） |

- **普通团子**："向前"即正向（+1），"向后"即逆向（-1）
- **哈基布**："向前"即逆向（-1），"向后"即正向（+1），因为其行进方向与普通团子相反

## 团子与技能

### 普通团子（6名）

| 团子 | 技能 | 触发条件 |
|------|------|----------|
| **奥古斯塔** | 本回合不行动，下回合最后行动 | 回合开始时处于堆叠最顶端 |
| **尤诺** | 将排名前后的其他团子传送至自己所在格 | 每场一次，经过赛程中点（pos ≥ 16）后触发 |
| **弗诺诺** | 移动时额外前进3格 | 回合开始时处于堆叠最底层 |
| **长离** | 65%概率下回合最后行动 | 下方堆叠有其他团子 |
| **今汐** | 40%概率移至格子最顶部 | 头顶堆叠有其他团子 |
| **卡卡罗** | 移动时额外前进3格 | 开始移动时为最后一名 |

### 特殊团子：哈基布

- 起始于31号位，逆向移动，掷骰1-6
- 第4回合起加入行动序列
- 不参与最终排名结算
- 若终点遇到正向普通团子，永远堆叠于最底部
- 若整轮中任何普通团子位置超过哈基布，哈基布传送回31号位

## 排名规则

1. 正向累计进度（`fwd`）越高，排名越靠前
2. 同格内堆叠越靠上，排名越靠前
3. 哈基布不计入排名

## 技术实现

### 核心优化

- **Numba JIT**：`@njit(cache=True)` 将核心模拟编译为机器码，单核约 15-20万局/秒
- **链表式棋盘**：`tile_bot` / `above` / `below` 数组实现 O(1) 插入删除
- **SplitMix64 内联 RNG**：所有随机数通过 `_rng_*` 内联函数生成，零开销
- **多进程并行**：`multiprocessing.Pool` 分 batch 并行，实时显示进度

### 置信区间公式

采用 **Wilson score interval**（比正态近似在小样本下更准确）：

```
Z = 2.5758 (99%)
center = (p̂ + Z²/2n) / (1 + Z²/n)
margin = Z × √(p̂(1-p̂) + Z²/4n) / n  /  (1 + Z²/n)
CI = [center - margin, center + margin]
```

## 规则歧义与假设

以下规则存在歧义，本实现采用如下假设：

1. **哈基布加入时机**："第三回合后"解释为**第4回合开始加入**
2. **绿色/红色连锁**：特殊点位效果**仅触发一次**，不连锁
3. **尤诺"排名前后"**：理解为当前排名中**紧邻尤诺的前一名和后一名**（如无则只传存在的）
4. **尤诺传送堆叠**：被传送团子按传送前排名顺序放置于尤诺所在格**顶部**
5. **多团子最后行动冲突**：奥古斯塔和长离均指定最后行动时，在行动序列末尾**随机排列**
6. **今汐技能时机**：在**回合开始阶段**判定（与其他 pre-round 技能一同处理）
7. **长离技能时机**：在**回合开始阶段**检查下方堆叠并决定是否标记最后行动
8. **黑色点位重排**：只重排普通团子，哈基布（如在）保持在最底部不动
9. **哈基布与普通团子相遇**：哈基布置于最底部，**不会被普通团子移动带动**
10. **绿色/红色对哈基布**：按哈基布行进方向解释"向前/向后"（与普通团子相反）
11. **堆叠带动规则**：移动时只带动自身及上方的团子，下方团子不受影响
