# 团子赛跑模拟 — Group A（淘汰赛Ver.）

基于 numba JIT + 链表棋盘 + 多进程并行对团子赛跑游戏进行高性能蒙特卡洛模拟，计算每个普通团子的胜率、平均名次、前2名/前3名概率及 99% 置信区间。

## 快速开始

```bash
# 默认 1,000,000 次模拟
python simulator.py

# 自定义模拟次数和并行进程
python simulator.py 5000000 --workers 16

# 单局详细调试模式
python simulator.py --test

# 多局调试
python simulator.py --test --test-n 5
```

依赖：Python 3.10+, numpy, numba

```bash
pip install numpy numba
```

## 游戏地图

32 个点位（0–31），环形排列。普通团子从 **0 号位** 正向移动（0→1→…→31），哈基布从 **31 号位** 逆向移动（31→30→…→0）。

### 特殊点位

| 类型 | 点位 | 效果 |
|------|------|------|
| 绿色 | 3, 9, 19 | 该格上所有团子整体向前移动 1 格（含堆叠团子） |
| 红色 | 15, 25, 29 | 该格上所有团子整体向后移动 1 格（含堆叠团子） |
| 黑色 | 5, 13, 22 | 随机重排该格上所有普通团子的堆叠顺序（不含哈基布） |

> 特殊格效果不连锁触发——每次落点只处理一次。

## 团子与技能

### 普通团子

| 团子 | 技能 | 触发条件 |
|------|------|----------|
| **奥古斯塔** | 本回合不行动，下回合最后行动 | 回合开始时处于堆叠最顶端 |
| **尤诺** | 将排名前后各一名的团子传送至自己所在格（每场一次） | 经过赛程中点（fwd ≥ 16）后 |
| **今汐** | 40% 概率移至当前格堆叠最上方 | 头顶有其他团子 |
| **卡卡罗** | 移动时额外前进 3 格 | 开始移动时为最后一名 |
| **绯雪** | 之后每次行动额外前进 1 格 | 与哈基布同格相遇后永久生效 |
| **卡提希娅** | 之后每次行动 60% 概率额外前进 2 格（每场最多解锁一次） | 移动结束后处于最后一名时解锁 |

### 特殊团子：哈基布

- 起始于 31 号位，逆向移动，掷骰 1–6
- 第 4 回合起加入行动序列
- 不参与最终排名结算
- 特殊格效果对哈基布同样生效
- 若终点遇到正向普通团子，永远堆叠于最底部
- 若整轮中任何普通团子位置超过哈基布，哈基布传送回 31 号位

### 简要分析

- **卡提希娅** 胜率最高（23.93%）、平均名次最好（2.89）、前3名概率最高（63.99%）。解锁后每回合 60% 概率 +2 格的持续收益非常可观
- **今汐** 紧随其后（胜率 22.65%），40% 概率跳至堆叠顶层可有效规避被底层团子拖累的风险
- **绯雪** 和 **卡卡罗** 位列中游，两者都有条件性的额外步数加成，表现稳定
- **尤诺** 的一次性传送技能不足以与持续性被动技能竞争（胜率 11.80%）
- **奥古斯塔** 表现最差（胜率仅 4.90%，末位率 59.29%）。处于堆叠顶端时跳过回合的惩罚过于严厉——顶端位置往往意味着领先，但跳过回合反而让其他团子追赶上来

## 规则歧义与假设

| # | 歧义 | 本实现采用的解释 |
|---|------|-----------------|
| 1 | 终点判定 | linear distance ≥ 31，允许越过 31 号点 |
| 2 | 特殊格连锁 | 不连锁触发，每次落点只处理一次 |
| 3 | 哈基布加入时机 | "第三回合后" → 第 4 回合起加入行动序列 |
| 4 | 奥古斯塔"最后行动" | 整个行动序列的最后（包括哈基布之后） |
| 5 | 尤诺传送目标 | 传送排名紧邻尤诺前一位和后一位的团子（最多 2 人） |
| 6 | 尤诺传送堆叠 | 被传送团子按原排名从低到高依次放置，使排名较高者靠上 |
| 7 | 卡提希娅"+2格" | 加在基础骰子结果之上（即实际移动 = 骰子 + 2） |
| 8 | 黑色点位重排 | 只重排普通团子，哈基布保持在最底部 |
| 9 | 哈基布移动带动 | 哈基布始终在堆叠最底部，移动时不会带动普通团子 |

## 技术架构

- **numba JIT**：核心模拟循环 `@njit(cache=True)` 编译为机器码，单核约 30 万局/秒
- **链表式棋盘**：`tile_bot` / `above` / `below` 数组实现 O(1) 堆叠插入与删除
- **SplitMix64 RNG**：内联零开销伪随机数生成器
- **多进程并行**：`multiprocessing.Pool` 分 batch 并行，8 核约 330 万局/秒
- **Wilson 分数区间**：99% 置信水平
